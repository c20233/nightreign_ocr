# ELDEN RING: Night Reign - Relic OCR Analyzer

『ELDEN RING: Night Reign』のインベントリ動画から、所有している**遺物（Relic）の詳細ステータスをOCRで自動抽出し、JSON形式で一括出力するツール**です。

---

## 🛠️ 主な機能

* **高精度スキャン (99.8%+)**: 画像変化検知により動画内のスクロール切り替えを特定し、表示が最も安定している「中央フレーム」のみをOCRスキャンします。
* **表記揺れ自動補正**: レーベンシュタイン距離アルゴリズムを用いてOCRの誤認識を補正。通常遺物（24パターン）およびWiki準拠の固有遺物（45種類）に対応。
* **ノイズフィルタリング**: メリット/デメリットのアイコン誤読（`点`、`志`等）や極小の不要行を自動クリーニング。
* **ワンクリック動作**: Windows環境向けに、環境構築・実行用およびアンインストール（クリーンアップ）用のバッチファイルを同梱。
* **GPUアクセラレーション**: NVIDIA製GPU（CUDA）による高速解析に対応（約2分間の動画を約50秒で処理）。

---

## 📋 動作要件

* **OS**: Windows 10 / 11 (64-bit)
* **Python**: 3.10 〜 3.12 (64-bit)  ※インストール時に「Add Python to PATH」へのチェックが必須。
* **GPU（任意）**: NVIDIA製グラフィックボード (GPU版を利用する場合のみ)

---

## 💻 使用手順

1. **ダウンロードと配置**
   * Releases から `relic_ocr_dist.zip` をダウンロードし解凍します。
   * 解析対象の動画ファイルをフォルダ内に配置し、ファイル名を **`game_play_record.mp4`** に変更します。
2. **実行**
   * **`setup_and_run.bat`** をダブルクリックして起動します。
   * 初回のみ、インストールするバージョンを以下から選択します：
     * `1`: CPU版（グラフィックボード非搭載PC用）
     * `2`: GPU版（NVIDIA製グラフィックボード搭載PC用）
   * セットアップ完了後、自動的に解析が開始され、同フォルダ内に **`game_stats.json`** が出力されます。
3. **アンインストール（クリーンアップ）**
   * ツールが不要になった場合は、以下の手順を実行してください。
     1. フォルダ内の **`cleanup.bat`** をダブルクリックします。
        （構築された仮想環境「venv」や一時ファイルが自動的に削除されます）
     2. その後、本フォルダ自体をごみ箱に捨てることで、PC上からのアンインストールが完全に完了します。
