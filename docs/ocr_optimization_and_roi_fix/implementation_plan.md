# OCR最適化およびROI動的化・調整

本プロジェクトでは、ゲームのプレイ動画からテキスト情報を抽出する処理の高速化および精度向上を目的とした以下の修正を行います。

## 背景
1. 1フレームごとにOCRを実行していたため、処理時間がかかっていた。
2. ソース動画の解像度が異なるとROI（抽出エリア）がずれる問題があった。
3. デメリット効果などを誤って認識するアイコンが含まれていたため、テキストが正しく抽出されない（特にframe0のタイトル・効果など）問題があった。

## Proposed Changes

### [MODIFY] [main.py](file:///c:/Users/3950x/Documents/python/nightrain_ocr/main.py)
- **バッチ処理の導入**: フレームごとの処理から、変化があったフレームをリスト化し、4フレームずつ `PaddleOCR` に渡すことでOCR回数を削減し処理を高速化します。
- **解像度非依存のROI計算**: 固定座標ではなく、フレームの幅(w)と高さ(h)に対する割合を用いてROI（抽出エリア）を計算するように変更します。
- **抽出エリア（x1）の調整**: 不要なアイコンが文字として認識されないように、抽出開始位置（x1）の割合を微調整し、純粋なテキストのみをOCRエンジンにかけるように修正します。

## Verification Plan
1. スクリプトを実行し、エラーが発生せずに `game_stats.json` が生成されることを確認する。
2. `game_stats.json` の内容（特に `frame: 0` のアイテム名や効果テキスト）がサンプル画像と一致し、不要な文字やアイコンが混じっていないことを確認する。
3. メリットとデメリットの色判別が正しく行われ、テキストが各属性に割り当てられていることを確認する。
