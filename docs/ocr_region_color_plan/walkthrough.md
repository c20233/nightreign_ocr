# 実装完了の確認: OCR領域分割と文字色判定

ご指示いただいた内容に基づき、`main.py` の改修を完了いたしました。

## 実装した機能
1. **4つの領域へのOCR分割**
   前回の超高速化（画面の差分検知）を利用し、画面が切り替わったと判定されたタイミングで、ご指定の「タイトル、効果1、効果2、効果3」の4つの座標枠に対して個別にOCRを実行するように変更しました。

2. **色の自動判定によるメリット・デメリットの分類**
   `classify_text_color` 関数を実装しました。読み取った効果枠の「もっとも明るい文字部分の色合い」を判定し、彩度が低い白文字であれば `"merit"`、彩度が高く色相が水色～青の範囲であれば `"demerit"` と判断します。

3. **JSON出力構造のアップデート**
   抽出結果のJSONが以下のフォーマットで出力されるようになりました。
   ```json
   {
       "frame": 60,
       "stats": {
           "title": "...",
           "effect1": {
               "type": "merit",
               "text": "..."
           },
           ...
       }
   }
   ```
