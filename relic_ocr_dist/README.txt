===================================================
  Night Reign OCR Analyzer - ユーザーマニュアル
===================================================

このツールは、ゲーム『ELDEN RING: Night Reign』等の動画から
遺物（ relic ）の詳細ステータスをOCRで自動抽出し、JSON形式で出力するツールです。

---------------------------------------------------
■ 動作要件 と 必要ディスク容量
---------------------------------------------------
- Windows 10 / 11 (64-bit)
- Python 3.10 〜 3.12 (64-bit)
  ※インストール時に「Add Python to PATH」に必ずチェックを入れてください。
- NVIDIA GeForce などのグラフィックボード (GPU版を利用する場合のみ)

【セットアップ後の仮想環境（venv）のディスク占有量】
- CPU版を選択した場合 : 約 1.5 GB 〜 2.0 GB
- GPU版を選択した場合 : 約 5.3 GB （各種CUDA/cuDNNランタイムを内包するため）
  ※ディスク空き容量に余裕があることを事前にご確認ください。
----------------------------------------------------
■ 使い方 (初回セットアップ ＆ 実行)
---------------------------------------------------
1. このフォルダ内に、解析したい動画ファイルを配置し、
   ファイル名を「game_play_record.mp4」に変更します。

2. 「setup_and_run.bat」をダブルクリックして起動します。

3. 初回起動時のみ、インストールするバージョンを選択する画面が表示されます。
   - [1] CPU版 (グラフィックボードを搭載していないPC用)
   - [2] GPU版 (NVIDIA製グラフィックボードを搭載しているPC用)
   ※ご自身のPC環境に合わせて 1 または 2 を入力して Enter を押してください。

4. 自動的に環境構築が始まり、必要なライブラリがインストールされます。
   (GPU版の場合は大容量のダウンロードが発生するため、数分かかります)

5. インストール完了後、自動的に動画のOCR解析が開始されます。

6. 解析が完了すると、同フォルダ内に「game_stats.json」が出力されます。
   ※2回目以降の起動時は、セットアップはスキップされて即座に解析が始まります。

---------------------------------------------------
■ クリーンアップ (アンインストール)
---------------------------------------------------
ツールの使用を終え、PC内の容量を空けたい場合は、以下の手順を実行してください。

1. 同フォルダ内の「cleanup.bat」をダブルクリックします。
   （構築された仮想環境「venv」や一時ファイルが自動的に削除されます）

2. その後、本フォルダ「relic_ocr_dist」自体をごみ箱に捨てることで、
   PC上からのアンインストールが完全に完了します。
